## UDP based Python Project

This project is a team project that will run a laser tag game, it will run with each character having a gun, body armour, as well as a helmet. There will be two teams red and green. The gun will be able to shoot the helmet or armour of the other team for points, as well as a tower, and the game will run for 6 minutes.

* Create a splash screen 
* Create a player entry screen of 15 players
* Create a screen for game progress
* Use UDP SOCKETS to broadcast and receive equipment codes for each player
* Link the database to the application for communication
* Run the game

## Running a Traffic Generator

In the ___ directory is a file named traffic_generator.py. This file generates UDP traffic so that you can simulate the game without the hardware. To run use this command
$ python src/traffic_generator.py.
